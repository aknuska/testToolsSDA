Tytuł: Czy można stworzyć rezerwację bez podania nazwiska


Preconditions:
- server dostepny


Kroki:
1. Wyslij poprawne zapytanie dla rezerwacji bez podania nazwiska:

Metoda: POST 
URI: https://restful-booker.herokuapp.com/booking
Heather: Content-Type: application/json  
Body
{
    "firstname" : "<name>",
    "lastname" : "",
    "totalprice" : <price>,
    "depositpaid" : <true/false>,
    "bookingdates" : {
        "checkin" : "<yyyy-mm-dd>",
        "checkout" : "<yyyy-mm-dd>"
    },
    "additionalneeds" : "text"
}

Expected:
Rezerwacja nie została stworzona pomyslnie:

StatusCode: 400 not ok
